2. EventBus Architecture Pub/sub wiring

All market data events flow through a single in-process EventBus. Publishers (client manager workers, user data stream) emit typed frozen events. Subscribers (strategy hooks, portfolio) receive them sequentially with backpressure.

5. Multiplex Socket Architecture Market data ingestion
